# Macon County Website Redesign

This project is a responsive redesign prototype for the Macon County, NC website created for WEB 210.

## Project Overview

The goal of this project was to create a user-centered, accessible, responsive website for Macon County. The site focuses on clear navigation, fast loading, mobile-friendly design, and a professional local government visual style.

## Pages Included

- Home
- Government
- Services
- Community
- Business
- Visit
- Contact

## Features

- Responsive Bootstrap-based layout
- Two-level navigation
- Homepage carousel
- Section banner images
- Services table
- Community event cards
- Contact form
- Optimized images
- Accessibility-focused structure

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
